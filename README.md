This is a Linux shell written from scratch in C. 
Pronounciation: "Grinch" (as in "How the Grinch Stole Christmas").

Current supported features:
- Piping
- File I/O redirection with ```<```, ```>```, and ```>>```
- Process signal handling

Builtins:
- ```cd```
- ```exit```

To use, run ```make```, then ```./grnsh```.
